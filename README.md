# Euro Finance (WERC) — Smart Contracts

Official smart contract repository for Euro Finance (WERC).

---

## ⚠️ Important Disclaimer

WERC is NOT:
- a stablecoin  
- pegged to EUR  
- affiliated with Circle or EURC  

WERC is a fully market-driven DeFi asset.
